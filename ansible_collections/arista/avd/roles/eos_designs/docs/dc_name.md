# dc_name

## Description

Optional DC Name, only used in SNMP location and Fabric Documentation.<br>Recommended to be common between all devices within a Site<br>

| -------- | ---- | -------- | ------- | ------------------ | ----------- |
| dc_name | String |  |  |  | DC Name |
